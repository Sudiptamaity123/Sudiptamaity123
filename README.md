## 🚀 Featured Projects

### 🤖 MyOwnAI — Vector Database & Local RAG System
> `C++` `HNSW` `KD-Tree` `Ollama` `REST API` `PCA`

A vector database built **from scratch in C++** with a full local RAG pipeline.

- Implemented **3 search algorithms** (HNSW, KD-Tree, Brute Force) and **3 similarity metrics** over 768-dimensional embeddings
- Achieved **12–15× query speedup** vs brute-force via HNSW ANN indexing with <5% recall loss
- Engineered a full RAG pipeline using Ollama (`nomic-embed-text` + `Llama 3.2`) — **>85% retrieval precision** on a 500-document corpus
- Exposed **5 REST APIs** + PCA-based web dashboard for real-time benchmarking; **<20ms p99** latency on 100K+ embeddings
